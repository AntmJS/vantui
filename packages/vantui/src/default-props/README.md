# DefaultProps 默认参数配置

### 介绍

全局为组件提供默认参数，如设置图片的 loading 占位、图片 error 占位、Toast 的展示时长 duration 等属性，页面 props 可以再将其覆盖

### 引入

在 Taro 文件中引入组件

```js
import { DefaultProps } from '@antmjs/vantui'
```

## 代码演示

### 基本用法

- 通过 `set` 方法设置图片的默认值
- 建议在入口文件使用，则不需要赋值`originProps`来还原
