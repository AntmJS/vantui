# WaterMark 水印

### 介绍

页面上添加特定的文字或图案, 作为版权标识或者其他特定信息标识

### 引入

在 Taro 文件中引入组件

```js
import { WaterMark } from '@antmjs/vantui'
```

## 基本使用

```jsx
function Demo() {
  const [show, setShow] = react.useState(1)

  return (
    <WaterMark content="@antmjs/vantui" fontSize={18} width={120} height={90} />
  )
}
```
