# ellipsis

### 介绍

展示文本内容过多的时候，隐去部分内容并用“…”替代，可以自定义收起内容后的行数，可以手动控制内容是否全部展示

### 引入

在 Taro 文件中引入组件

```jsx
import { Ellipsis } from '@antmjs/vantui'
```
